# 🌐 Web-Development-AH

Welcome to **Web-Development-AH**!
This repository contains all class codes and learning materials related to web development.

---

## 📚 About

This repository is created to store and share code examples from web development classes. It covers multiple technologies from basic to intermediate level to help students learn and practice efficiently.

---

## 🛠️ Technologies Included

* **HTML** – Structure of web pages
* **CSS** – Styling and layouts
* **JavaScript (JS)** – Interactivity and logic
* **Bootstrap** – Responsive design framework
* **PHP** – Backend development
* **MySQL** – Database management

## 🚀 How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/Web-Development-AH.git
   ```

2. Open any folder based on the topic.

3. Run files in:

   * Browser (HTML, CSS, JS)
   * Local server (PHP, MySQL)
